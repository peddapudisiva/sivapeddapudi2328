## 🌟 High-Impact Features

1. **🤖 Interactive "Ask Siva AI" Chatbot Widget**
   - Built-in glassmorphism AI chat assistant loaded with Siva's complete knowledge base (projects, skills, internships, education). Recruiter Q&A supported.

2. **📄 Direct PDF Resume Download**
   - Single-click resume download button directly accessing [`/siva_resume.pdf`](public/siva_resume.pdf).

3. **🏆 Hackathons & Industry Certifications Showcase**
   - Features **2nd Place Freedom Fest 2026 Vibe Coding Hackathon** win, **2nd Prize National Idea Hackathon**, **Global STEM Expo**, and 8+ certifications (Anthropic AI, Google Gemini, Google Cloud Arcade, IBM Data Science, AWS, TATA, Deloitte).

4. **💻 3D Interactive Project Showcase**
   - **ParticleVerse Studio**: 3D particle engine rendering 100,000 particles at 60 FPS with Three.js & R3F.
   - **Jarvis AI Assistant**: Desktop voice AI assistant using Speech Recognition & LangChain.
   - **Vahaan Bazar**: Vehicle marketplace ML recommendations with 80%+ resale prediction accuracy.

5. **📝 AI Technical Writeups & Articles**
   - Technical MDX articles covering WebGL shaders, voice AI agents, and n8n workflow automation.

## 🚀 Local Installation & Setup

```bash
# Clone the repository
git clone https://github.com/peddapudisiva/portfolio.git

# Navigate into directory
cd portfolio

# Install dependencies
npm install

# Run development server
npm run dev

# Build production bundle
npm run build
```
